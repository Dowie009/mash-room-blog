# 写真サイズの変更方法（シンプル版）

## ファイル
`src/components/HorizontalScroll.astro`

## 変更する場所

### 📸 チャプター1（Story Panel）
**30行目**のこの部分を変更：
```html
h-[30vh] md:h-[70vh]
```

### 📸 チャプター2（Gear Panel）
**60行目**のこの部分を変更：
```html
h-[60vh] md:h-[70vh]
```

### 📸 チャプター3（Tech Panel）
**86行目**のこの部分を変更：
```html
h-[60vh] md:h-[70vh]
```

## 変更方法

**`h-[30vh]`** の数字部分だけを変更してください。

- `h-[10vh]` = 画面の20%（小さく）
- `h-[30vh]` = 画面の30%
- `h-[50vh]]` = 画面の50%
- `h-[70vh]` = 画面の70%（大きく）

**`md:h-[70vh]`** はデスクトップ用なので、同じ数字に変更してください。

## 例

### 小さくする場合
```html
<!-- 変更前 -->
h-[30vh] md:h-[70vh]

<!-- 変更後 -->
h-[20vh] md:h-[20vh]
```

### 大きくする場合
```html
<!-- 変更前 -->
h-[30vh] md:h-[70vh]

<!-- 変更後 -->
h-[50vh] md:h-[50vh]
```

## 注意
- `aspect-[3/4]`は変更しないでください（アスペクト比は固定）
- 変更後はブラウザをリフレッシュ（F5）してください
