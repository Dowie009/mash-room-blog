# HorizontalScroll.astro への統合ガイド

ジェミニから受け取ったHTMLコードを安全に統合する手順です。

---

## ⚠️ 重要な確認事項

### 1. HTML形式の確認
ジェミニから受け取ったコードが：
- **Astro形式**（`.astro`）になっているか
- それとも**純粋なHTML**か

### 2. スクリプト部分の確認
既存のHorizontalScroll.astroには：
- GSAPアニメーションのスクリプトが含まれている
- これを維持する必要がある

---

## 🔄 安全な統合手順

### ステップ1: バックアップ
```bash
# 既存ファイルをバックアップ
cp src/components/HorizontalScroll.astro src/components/HorizontalScroll.astro.backup
```

### ステップ2: 受け取ったコードの確認

#### パターンA: Astro形式の場合
```astro
---
// フロントマター（あれば）
---

<!-- HTML部分 -->
<section>...</section>

<style>
  /* スタイル */
</style>

<script>
  // スクリプト
</script>
```

#### パターンB: 純粋なHTMLの場合
```html
<section>...</section>
<style>...</style>
<script>...</script>
```

### ステップ3: 統合方法

#### 方法1: 完全置き換え（推奨）
**条件**: ジェミニが完全なAstroコンポーネントを提供した場合

1. 既存の `HorizontalScroll.astro` を開く
2. **全体を選択して削除**
3. ジェミニのコードを**そのまま貼り付け**
4. 画像パスを確認（`/images/` で始まっているか）

#### 方法2: 部分置き換え
**条件**: HTML部分だけを受け取った場合

1. 既存の `<section class="horizontal-wrapper">` から `</section>` までを選択
2. ジェミニのHTML部分で置き換え
3. `<style>` セクションを更新
4. **`<script>` セクションは既存のものを維持**（アニメーションが重要）

---

## 📋 チェックリスト

統合前に確認：

- [ ] 画像パスが `/images/` で始まっている
- [ ] 既存の画像ファイル名と一致している
- [ ] `<script>` タグが含まれている（アニメーション用）
- [ ] スタイルが `<style>` タグ内に含まれている
- [ ] Astro形式になっている（`---` フロントマターがあればOK）

統合後：

- [ ] ブラウザで表示を確認
- [ ] アニメーションが正常に動作する
- [ ] 画像が正しく表示される
- [ ] エラーがないか確認

---

## 🔍 よくある問題と解決方法

### 問題1: スクリプトが重複している
**解決**: 既存のスクリプトを削除して、ジェミニのスクリプトを使用

### 問題2: 画像パスが違う
**解決**: すべての画像パスを `/images/` 形式に統一

### 問題3: スタイルが適用されない
**解決**: `<style>` タグが正しく閉じられているか確認

---

## 💡 推奨手順（最も安全）

1. **既存ファイルをバックアップ**
2. **ジェミニのコードを新しいファイルに保存**（例: `HorizontalScroll.new.astro`）
3. **内容を確認**
4. **問題なければ置き換え**

---

## 📝 確認したいこと

ジェミニから受け取ったコードを教えてください：
- Astro形式ですか？それともHTML形式ですか？
- `<script>` タグは含まれていますか？
- 画像パスは `/images/` で始まっていますか？

これらの情報があれば、より具体的な手順を案内できます！
