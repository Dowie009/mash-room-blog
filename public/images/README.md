## ✏️ 画像の差し替え方法

### 簡単な方法（ファイル名そのまま）

1. 新しい画像を用意
2. 既存のファイル名にリネーム
3. このフォルダに上書き保存

**例:** `Blog_00.png` を差し替える
```
新しい画像を「Blog_00.png」にリネーム
↓
public/images/Blog_00.png に上書き
↓
完了！（コード変更不要）
```

### 新しいファイル名で追加する場合

1. 新しい画像をこのフォルダに配置
2. `CONTENT.md` を編集してパスを変更
3. `npm run update-content` を実行

**詳細は**: `docs/IMAGE_GUIDE.md` を参照
